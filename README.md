# ✈️ Global Flights Data Pipeline

A complete data engineering pipeline using open-source flight data from Kaggle. This project covers:

- Data ingestion with Python
- Data modeling and transformation using dbt
- Workflow orchestration with Apache Airflow
- PostgreSQL as the data warehouse
- Monitoring with Prometheus + Grafana
- Infrastructure setup with Docker

## Dataset
- Source: Kaggle - US Flight Delays 2015
- Sample: 100,000 flights
